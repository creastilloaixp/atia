# 🧠 Base de Conocimiento ATIA (Obsidian Integration)

Esta carpeta contiene la información oficial de Grupo ATIA Inmobiliaria.
Todo el contenido aquí es leído automáticamente por **Adriana IA** para dar respuestas precisas.

## ¿Cómo funciona?
1. Edita o crea archivos `.md` en esta carpeta
2. Haz commit y push a GitHub
3. Adriana absorbe la información automáticamente en segundos

## Estructura
```
knowledge/
├── Ventas/           → Proceso, requisitos y comisiones de venta
├── Rentas/           → Requisitos, reglas y horarios de renta
├── Inversiones/      → Tipos de inversión y zonas premium
├── Valuacion/        → Métodos y costos de avalúo
└── General/          → Información general de ATIA
```

## Reglas
- Usa formato **Markdown** (.md)
- Un archivo por tema específico
- Nombra las carpetas según el servicio/tema
- Adriana usará esta información para responder a clientes en WhatsApp y en la web
