# Cinematic Modules Pack

11 modular HTML/CSS/JS components for building premium cinematic websites.
Each module is self-contained — copy the code into any project.

## Modules

| # | Module | Description | Best For |
|---|--------|-------------|----------|
| 01 | **Scroll Video Player** | Maps video frames to scroll position | Hero sections, product reveals |
| 02 | **Accordion Slider** | Expand-on-hover image panels | Menus, portfolios, categories |
| 03 | **Kinetic Text** | Character-by-character scroll reveal | Headlines, brand statements |
| 04 | **Reveal Text** | Slide-up masked text animation | Section titles, stories |
| 05 | **Typewriter** | Character typing with cursor | Taglines, code blocks |
| 06 | **Glitch Effect** | Cyberpunk-style distortion | Bold headers, tech brands |
| 07 | **Image Trail** | Cursor leaves image trail | Creative portfolios, shops |
| 08 | **Flip Cards** | 3D hover flip cards | Services, team, testimonials |
| 09 | **SVG Draw** | Path drawing on scroll | Logos, icons, decorative |
| 10 | **Parallax Hero** | Multi-layer depth scroll | Hero sections, intros |
| 11 | **Smooth Scroll Nav** | Fixed nav with section tracking | Single-page navigation |

## Usage in Agent SKILL

The cinematic-sites agent skill references these modules by number.
When building a website, the agent selects appropriate modules and
integrates them into the final HTML build.

## Quick Start

Open any `.html` file directly in a browser to preview the module.
