Kommo CRM (antes amoCRM)

Para crear leads automaticamente en tu pipeline de ventas

Requiere cuenta activa

Cada vez que un visitante completa el formulario de tu landing, el sistema crea automaticamente un nuevo contacto y lead en tu cuenta de Kommo con toda la informacion capturada (nombre, telefono, situacion, ubicacion, urgencia).

Dato que necesitamos Ejemplo Prioridad
Subdominio de Kommo tuempresa.kommo.com Obligatorio
API Token (Long-lived) eyJhbGciOi...xxxxx Obligatorio
ID del campo "Telefono" 123456 (numerico) Obligatorio
Pipeline ID (embudo de ventas) 7890123 Opcional
ID del responsable (usuario asignado) 456789 Opcional
🔎 Como obtener estos datos paso a paso
  1. Inicia sesion en https://tuempresa.kommo.com con una cuenta de administrador
  2. Ve a Configuracion (icono de engrane, esquina inferior izquierda)
  3. Haz clic en Integraciones → API
  4. Crea una nueva integracion o usa un token existente — necesitas un "Long-lived token"
  5. Copia el token completo (es un texto largo que empieza con "eyJ...")
  6. Para el ID del campo Telefono: ve a Configuracion → Campos → Contactos, busca "Telefono" y anota el numero ID que aparece
  7. Para el Pipeline ID: ve a Configuracion → Embudos de venta, haz clic en el embudo deseado y copia el ID de la URL
📞

Zadarma (Telefonia VoIP)

Para llamar automaticamente al asesor cuando entra un lead

Requiere cuenta activa

Segundos despues de que un lead completa el formulario, Zadarma genera un callback automatico: primero llama al asesor de ventas, y cuando contesta, conecta la llamada con el lead. El resultado: tu equipo contacta al cliente en menos de 60 segundos, cuando su interes esta al maximo.

Dato que necesitamos Ejemplo Prioridad
API Key de Zadarma abc123def456... Obligatorio
API Secret de Zadarma xyz789ghi012... Obligatorio
Numero de origen (PBX/SIP) 100 o 5512345678 Obligatorio
Extension del asesor de ventas 101 Opcional
🔎 Como obtener estos datos paso a paso
  1. Inicia sesion en https://my.zadarma.com
  2. Ve a Configuracion → API → Claves de acceso
  3. Si no tienes claves, haz clic en "Generar nuevas claves"
  4. Copia el API Key (campo superior) y el API Secret (campo inferior)
  5. Para el numero de origen: ve a Mi PBX → Extensiones y copia el numero o extension que recibirá las llamadas primero (tu asesor de ventas principal)
⚠️ Importante: En Zadarma, ve a Configuracion → API → IPs permitidas y agrega la IP del servidor de automatizaciones. Sin esto, las llamadas seran rechazadas. Nosotros te proporcionamos la IP exacta.
